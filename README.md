# Tic Tac Toe

A simple Tic Tac Toe game built with React and Vite.
I found this old project from when I was learning React, so I thought I’d push it to GitHub.
## Features

- Two-player X and O gameplay
- Win detection and restart button
- Responsive board UI

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## GitHub Pages

This project uses a relative Vite base path so the production build works correctly when deployed to GitHub Pages.
